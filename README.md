## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/c4bae/handsim.git
   ```
2. Install NPM packages in the frontend directory
   ```sh
   npm install
   ```
3. Install python packages in the backend directory
   ```sh
   pip install -r requirements.txt
   ```
4. Run main.py and locally deploy the frontend

## Usage
Upon successful set up, your hand should now appear to be tracked within the 3D space. Use space while pointing at a block to grab it, and use space to release. Stay behind the green line when grabbing new blocks. Use w or s to move hand forwards or backwards.
